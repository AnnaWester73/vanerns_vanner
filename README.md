# 2 Koda vanerns_vanner
* Uppgift inlämnad av Anna Wester

## Systemet består av två huvuddelar
### Inventory och Excursion

#### Inventory funktioner
 * Lägga inventarier
 * Hyra ut inventerar
 * Hantera antal som finns kvar

#### Excursion funktioner
* Hantera medlemmar. Lägga till och ta bort.
* Hantering när medlem som hyr utrustning
* Hantering när medlem lämnar tillbaka utrustning
* Visa lista på medlemmar som inte lämnat tillbaka hyrd utrustning

## Projektstruktur
* Mappen scr består av funktioner och kod
* Mappen test består av alla testfall.

## Installation

* Tester utförs med pytest. Skriv py -m pytest i terminalen
